# AGRINHO_2026
Este projeto é um site educativo desenvolvido com HTML, CSS e JavaScript que tem como objetivo conscientizar sobre o desmatamento e seus impactos no meio ambiente.

A página apresenta de forma clara e organizada o que é o desmatamento, suas principais causas, consequências ambientais, sociais e econômicas, além de possíveis soluções para reduzir e reverter esse problema. Também inclui sugestões de ações individuais que ajudam na preservação das florestas.

O projeto foi criado com foco em educação ambiental, buscando incentivar hábitos mais sustentáveis e a reflexão sobre a importância da preservação das florestas para o equilíbrio do planeta.
